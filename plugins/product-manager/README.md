# product-manager

A Claude Code plugin with skills for end-to-end product management work — discovery, prioritization, roadmapping, and stakeholder communication.

## Status

Skeleton. Skills to be added under `skills/<name>/SKILL.md`.

## Local testing

```bash
claude --plugin-dir ./plugins/product-manager
```

Reload after edits with `/reload-plugins`. Validate with `claude plugin validate`.
